# Ketamine Therapy Finder

Find ketamine therapy clinics near you. Comprehensive directory of ketamine treatment providers for depression, PTSD, anxiety, OCD, chronic pain, and other treatment-resistant conditions.

## Tech Stack
- Next.js 14 (App Router)
- Supabase (PostgreSQL)
- Stripe (payments)
- Tailwind CSS
- Deployed on Vercel
